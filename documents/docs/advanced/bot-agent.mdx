# Bot Agent

Bot Agent 是一种特殊的 Agent 角色，通过飞书或企业微信等外部渠道与用户交互。

## Bot Agent 的能力

- 接收用户在聊天工具中发送的消息
- 根据消息内容执行相应操作
- 返回执行结果给用户
- 支持内置斜杠命令

## 配置 Bot Agent

1. 创建一个 Agent 预设，角色选择「Bot」
2. 配置运行时和模型
3. 在项目设置的通知配置中绑定飞书/企微应用
4. 在 Bot Agent 配置中关联通知渠道

## 使用 Bot

配置完成后，在飞书或企业微信中找到你的 Bot 应用：

- 直接发送消息与 Bot 对话
- 使用 `/` 斜杠命令执行特定操作
- Bot 会将操作结果以消息形式返回

## 安全注意

Bot Agent 通过外部渠道暴露了部分系统操作能力，建议：

- 为 Bot Agent 配置较低的权限模式
- 限制 Bot Agent 可执行的操作范围
- 定期检查 Bot 的执行日志
