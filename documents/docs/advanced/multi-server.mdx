# 多服务器管理

Agent Spaces 前端支持同时连接多个后端服务器实例，方便管理不同环境的 Agent 服务。

## 使用场景

- **开发/生产隔离** — 开发环境和生产环境使用不同的服务器
- **团队协作** — 连接团队共享的 Agent Spaces 服务器
- **多项目管理** — 不同项目使用独立的服务器实例

## 配置方式

在登录页或顶部导航的服务器切换器中：

1. 点击服务器选择器
2. 添加新服务器（输入名称和地址）
3. 在多个服务器之间切换

每个服务器使用独立的 Secret Key 认证，切换服务器时需要重新登录。

## 数据隔离

不同服务器的工作空间、议题、任务、Agent 配置完全独立，互不影响。
