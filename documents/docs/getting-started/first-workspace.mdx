# 创建第一个工作空间

登录成功后，你会进入 Agent Spaces 的主页。接下来创建你的第一个工作空间。

## 什么是工作空间？

工作空间（Workspace）是你项目代码的容器。每个工作空间绑定一个本地目录，Agent 将在这个目录下读取和修改代码。

## 创建步骤

### 1. 点击创建工作空间

在主页点击「创建工作空间」按钮。

### 2. 选择代码目录

有两种方式绑定代码目录：

- **浏览本地文件夹**：点击文件夹图标，选择已有的项目目录
- **Git Clone**：输入仓库 URL，Agent Spaces 会自动拉取代码到本地

选择目录后，系统会自动在工作空间内创建 `.agentspace` 元数据目录，用于存储工作空间配置和 Agent 上下文。

### 3. 配置工作空间

为工作空间设置名称和描述，方便后续管理。

## 工作空间列表

创建完成后，你可以在主页看到所有工作空间。每个工作空间卡片显示：

- 工作空间名称和描述
- 绑定的代码目录路径
- 议题和任务数量
- 最近活跃时间

点击工作空间卡片即可进入 IDE 界面。
