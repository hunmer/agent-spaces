# 用量统计

Agent Spaces 内置用量统计仪表盘，帮助你了解 Agent 的 Token 消耗和费用情况。

## 首页仪表盘

登录后首页展示全局用量概览：

### 趋势图表

- **Token 消耗趋势** — 按天/周/月查看 Token 使用量变化
- **费用趋势** — 估算费用随时间的变化曲线
- **Agent 活跃度** — 各 Agent 的执行频率统计

### 模型统计

按 AI 模型分类展示：

- 每个模型的 Token 消耗量
- 每个模型的估算费用
- 调用次数统计

## 费用估算

系统根据各模型的定价自动估算费用。费用计算基于：

- 输入 Token 数量
- 输出 Token 数量
- 模型的单位价格

费用数据存储在本地 SQLite 数据库中，不会上传到外部服务。

## 数据来源

每次 Agent 执行任务时，系统自动记录：

- 执行的 Agent 名称和角色
- 使用的模型
- 输入/输出 Token 数
- 执行时间
- 估算费用

这些数据汇总后在仪表盘展示，帮助你优化 Agent 使用策略、控制成本。
