# 频道聊天

频道是工作空间内的沟通中心，你可以在这里与 Agent 进行对话。

## 频道与消息

每个工作空间包含多个频道，频道内可以发送消息、@mention Agent。

### 富文本编辑

消息编辑器基于 TipTap 富文本引擎，支持：

- **Markdown 格式** — 代码块、加粗、列表等
- **@mention** — 输入 `@` 触发 Agent 选择器，mention Agent 后会触发其执行
- **代码块** — 插入代码片段，支持语法高亮

### 消息类型

频道中的消息包含多种类型：

- 用户发送的文本消息
- Agent 的回复消息
- Agent 执行任务的进度更新
- 系统通知

## @mention 触发 Agent

在消息中输入 `@` 即可触发 Agent 选择器。选择 Agent 后：

1. 发送消息，被 mention 的 Agent 自动唤醒
2. Agent 根据消息内容执行相应操作
3. 执行结果在频道中实时展示

这种方式适合快速给 Agent 下达指令，不需要创建正式的议题。

## AI 消息渲染

Agent 的回复消息采用结构化渲染：

- **工具调用链** — 展示 Agent 的完整思考链路，每一步工具调用都清晰可见
- **工具详情** — 点击工具调用可展开查看输入参数、输出结果、代码 diff
- **中间输出** — Agent 的推理过程和中间结论
- **最终结论** — Agent 的最终回答或执行结果
