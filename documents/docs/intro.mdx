# Agent Spaces

Agent Spaces 是一个**本地多 Agent 协同编程平台**。它将 AI Agent 引入你的开发工作流，通过多种 Agent 角色实现任务的自动分发、代码修改、审核与提交。

## 为什么选择 Agent Spaces？

- **完全本地运行** — 代码不离开你的机器，数据存储在本地，安全可控
- **多 Agent 协同** — 调度者、策划者、执行者、审核者、提交者六种角色各司其职
- **IDE 级别体验** — 集成代码编辑器、终端、Git 操作、议题管理，一站式开发
- **多模型支持** — 支持 Claude Code、OpenAI Codex 等多种 AI 运行时
- **实时通知** — 通过飞书、企业微信接收 Agent 状态通知，远程操控

## 核心能力

| 功能 | 说明 |
|------|------|
| 工作空间管理 | 绑定本地代码目录，支持文件夹浏览和 Git Clone |
| 代码编辑器 | 基于 Monaco Editor 的完整编辑体验 |
| 终端 | 集成终端，可直接在平台内执行命令 |
| 议题管理 | 创建 Issue，Agent 自动分解为任务并执行 |
| 频道聊天 | @mention Agent 触发对话和任务执行 |
| Git 操作 | 可视化分支管理、提交、合并 |
| Agent 编排 | 六种 Agent 角色自动化协作 |
| 通知中心 | 飞书/企业微信消息推送和远程命令 |
| 用量统计 | Token 消耗、费用估算、趋势图表 |
